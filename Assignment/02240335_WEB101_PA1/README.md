# Spotify Clone - WEB101 Programming Assignment 1

## Project Overview

This project is a recreation of Spotify's web player interface using React and Tailwind CSS. The application demonstrates component-based architecture, responsive design principles, and modern front-end development techniques as required by WEB101 Programming Assignment 1.

### Original Application
**Spotify Web Player** - A music streaming platform's main interface featuring:
- Sidebar navigation
- Main content area with playlists
- Player controls
- Card-based layout for albums/playlists

## Features Implemented

### Core Functionality
- Responsive layout adapting to desktop, tablet, and mobile views
- Component-based architecture with reusable components
- Interactive UI elements (hover states, active states)
- Dynamic content rendering from data sources
- Mobile-first responsive design

### Components Created
1. **Sidebar** - Navigation menu with home, search, and library links
2. **ContentSection** - Main content area displaying playlists and albums
3. **Card** - Reusable component for displaying music items
4. **Player** - Bottom music player controls (static UI)
5. **Header** - Top navigation bar with user controls
